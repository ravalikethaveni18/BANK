# Django-chat-app
- This project,The Bank Management System is a secure and efficient banking solution allows users to manage their accounts, perform transactions. 
## Features
- create registration Form
- pin generation 
- deposit 
- checking Balance
- withdrawal
- account to account transfer
 ##languages
->python
->html,css
->django
->sqlite3

->Django==5.1.5
